SYBIL3
===========================================================================

Sybil is a Matlab-derived code and GUI that performs multivariate spatial analysis of 
scattered geological, geophysical and environmental data.

code version: 3.0 (2013)
programmer: L. Casini 
contact: casini@uniss.it
